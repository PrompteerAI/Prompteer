# Code Of Conduct

Prompteer follows the intent of the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) and adapts it for this project.

## Expected Behavior

- Be constructive, patient, and specific.
- Respect different experience levels, backgrounds, and viewpoints.
- Give feedback on the work, not the person.
- Own mistakes and correct them quickly.
- Keep public discussion appropriate for a professional open source project.

## Unacceptable Behavior

- Harassment, threats, stalking, or sustained disruption.
- Sexualized language or imagery.
- Insults, derogatory comments, personal attacks, or bad-faith participation.
- Publishing private information without explicit permission.
- Any conduct that would reasonably make the project unsafe or hostile.

## Scope

This code applies in project issues, pull requests, discussions, commits, reviews, documentation, chat spaces, and any public representation of the Prompteer project.

## Enforcement

Report conduct concerns privately through the repository maintainer or GitHub Security Advisories. Maintainers will review reports fairly, protect reporter privacy, and choose proportionate actions such as clarification, warning, temporary restriction, or permanent removal from project spaces.
